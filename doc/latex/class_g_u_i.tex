\doxysubsection{Detailed Description}
Manages the game\textquotesingle{}s graphical user interface. 

The \doxylink{class_g_u_i}{GUI} class handles the rendering and interaction components of the game\textquotesingle{}s user interface. It manages various textures, fonts, window settings, buttons, game state variables, and screens. This class processes user input, updates \doxylink{class_g_u_i}{GUI} states, and renders \doxylink{class_g_u_i}{GUI} components for different screens.
