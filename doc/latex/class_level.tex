\doxysubsection{Detailed Description}
Represents a game level with physics simulation. 

This class manages the elements and behavior of a game level, including Box2D physics simulation, game objects like birds, pigs, boxes, and walls, as well as handling the game\textquotesingle{}s progression and state.
